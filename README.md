### Instruções

> Recomendo Linux

```
gcc -c produto.c                              //Compila o arquivo módulo
gcc -c guluso.c ou tenta-erro.c               //Compila o arquivo principal
gcc -o exe produto.o guloso.o ou tenta-erro.o //Gera um executavel com os dois arquivos linkados
./exe                                         //Executa
```
